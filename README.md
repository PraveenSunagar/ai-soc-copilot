# ai-soc-copilot
# 🛡️ AI SOC Copilot Dashboard

A modern, responsive, and interactive **AI-powered Security Operations Center (SOC) Dashboard** built using **HTML, CSS, and JavaScript**. This project simulates a real-world SOC environment where security analysts can monitor alerts, analyze threats, visualize security metrics, and investigate incidents through an intuitive interface.

---

## 📌 Project Overview

The **AI SOC Copilot Dashboard** is designed to demonstrate the workflow of a Security Operations Center (SOC). It provides a professional cybersecurity dashboard interface featuring security alerts, AI-assisted analysis, MITRE ATT&CK mapping, threat intelligence lookup, and investigation timelines.

This project is intended for learning, portfolio demonstration, and frontend development practice in cybersecurity.

---

## 🚀 Features

* 📊 Professional SOC Dashboard UI
* 🚨 Security Alert Monitoring
* 📈 Alert Statistics & Analytics
* 🔍 Search Security Alerts
* 🎯 Severity Filtering (Critical, High, Medium, Low)
* 🤖 AI-Powered Alert Explanation
* 🛡️ MITRE ATT&CK Mapping
* 🌐 Threat Intelligence Lookup
* 📅 Investigation Timeline
* 🔔 Notification System

* 📉 Interactive Charts using Chart.js
